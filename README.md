# Deluge Magnet

Deluge Magnet is a lightweight MacOS app which handles forwarding magnet links to Deluge app.

## Requirements

- MacOS 10.13+
- Xcode 10.0

## Installation

1. Clone the repo
2. Build using Xcode
3. Run the app at least once to have it register itself for handling magnet links.
